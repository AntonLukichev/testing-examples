# testing-examples
Advanced testing with React Testing Library

![Pyramid of testing](https://miro.medium.com/max/704/1*R_tHYBM8myszwKa6vXfNbw.png "Pyramid of testing")

## Getting Started
Install dependencies
```shell script
yarn
```

Run tests
```
yarn test
```
